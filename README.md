### NFT Meta data: 
```javascript
{
    "name": "Image 0",
    "description": "Itachi",
    "symbol": "NFT",
    "image": "0.png",
    "seller_fee_basis_points": 100,
    "properties": {
        "files": [{ "uri": "0.png", "type": "image/png" }],
        "category": "image",
        "creators": [
            {
                "address": "CGuYmF72JEefCiAzNLTs6npffesS3aPAK5vRv4iKoLVS",
                "share": 100
            }
        ]
    }
}
```
### Clone Metaplex:

- git clone https://github.com/metaplex-foundation/metaplex.git
- cd metaplex/js
- yarn install
- yarn bootstrap
- yarn build
- yarn start
